Droit des mineurs et des majorités

Le Bright Bill

Accueil / Bright Bill

Le Bright Bill (BrBill) définit les étapes légales de la transition entre la minorité et la majorité, en accordant progressivement droits, libertés et responsabilités selon l'âge. Il a été promulgué et est en vigueur depuis le 14 mai 2026, 20:34. Règle d'interprétation : si un âge n'est pas mentionné pour une disposition, le règlement de l'âge précédent continue de s'appliquer jusqu'à la prochaine précision. Une émancipation totale dispense de tout accord parental requis ailleurs dans le texte.

Dispositions complémentaires

  • Salaire minimum jeunesse — obligatoire pour les travailleurs de 13 à 16 ans.
  • Accord parental — consentement signé d'un parent ou tuteur pour certains droits encadrés.
  • Accord parental étendu — consentement des deux parents ou tuteurs, sauf si le mineur est élevé par un seul.
  • Obligation d'un avocat — représentation obligatoire des mineurs et personnes vulnérables en procédure judiciaire.
  • Effacement automatique du casier pour infractions mineures, à partir de 18 ans.
  • Peines réduites pour les infractions commises avant 18 ans, selon la capacité de discernement.

Mobilisation militaire

Le service militaire est volontaire en temps normal. En cas de décision de l'autorité dirigeante, un code de mobilisation peut être activé :

CodePortéeDurée
Code K (minimal)Tous sexes1 an (6 mois actif + 6 mois réserve)
Code KS (standard)Hommes 18–21 ans, femmes 18–20 ans3 ans (H) / 2 ans (F)
Code KVS (neutre)Hommes 18–26 ans, femmes 18–23 ans3 ans (H) / 2 ans (F)
Code KE (difficile)Hommes 18–26 ans, femmes 18–23 ans5 ans (H) / 4,5 ans (F)
Code KVE (guerre)Hommes 16–28 ans (obligatoire), femmes 18–20 ans (volontaire)Jusqu'à la fin du conflit
Code KVES (extrême)Population 18–40 ans, tous sexesJusqu'à la fin du conflit ; mesures de protection familiale possibles
Code KVEV (état final)Population adulte, sans distinction de sexeJusqu'à la fin du conflit ; mineurs pris en charge par l'État

Les codes KVES et KVEV requièrent l'approbation majoritaire des généraux de guerre et du sénat. L'activation des autres codes relève exclusivement de l'autorité dirigeante.
